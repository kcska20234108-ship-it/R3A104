# **Streamlit Markdown 表現チュートリアル**

---

## **1. イタリックと太字 (Italics and Bold)**

Markdownでは、次のように文字装飾ができます。

- `_イタリック_` → _イタリック_  
- `**太字**` → **太字**  
- `**_太字かつイタリック_**` → **_太字かつイタリック_**

---

## **2. 見出し (Headers)**

見出しは、`#` の数で大きさを変えます。  
数が多いほど小さい見出しになります。
# 見出し1
## 見出し2
### 見出し3
#### 見出し4
##### 見出し5
###### 見出し6
